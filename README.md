# About The Project
This package provides a simulation for the Precision Farming Rover. 

![image](https://github.com/user-attachments/assets/0eb8b9d6-ee8e-4bc7-b435-e56494db9c2f)

To run this package, you'll need the following:

- Linux Ubuntu 24.04
ROS2 Jazzy Jalisco
Gazebo Harmonic
